# Reverse Tree

Implement a function to reverse a binary tree (i.e., flip it from right to left).

The function must be declared as follows:

``` c
void reverse_tree(struct s_node *root);
```

<br>

Example :  
``` c

	The following tree :

	       94
	       / \
	      /   \
	     34   52
	    / \
	   /   \
	  1    99
	 /     / \
	20    /   \
	     /     \
	    83     39
	     \     / \
	     61   /   \
	         37   67

	would become :

	      94
	      / \
	     /   \
	    52   34
	         / \
	        /   \
	       99    1
	      / \     \
	     /   \    20
	    /     \
	   39     83
	  / \     /
	 /   \   61
	67   37
```
